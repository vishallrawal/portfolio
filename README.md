# 🌌 Vishal Rawal Portfolio

A premium, modern, and highly responsive personal portfolio website showcasing my skills, projects, education, technical blog, and web development journey.

🔗 **Live Link:** [https://vishallrawal.github.io/](https://vishallrawal.github.io/)

---

## 🚀 Key Features

* ⚡ **Islands Architecture**: Built with Astro v5 and React 19 for optimal performance.
* 🎨 **Immersive Dark Theme**: Premium glassmorphic panels, glowing neon gradients, and mouse-following backdrop coordinate light rays.
* 🧩 **Dynamic Projects Showcase**: Showcases my real GitHub repositories with custom AI-generated graphic preview cards.
* ✍️ **Dynamic Tech Blog**: Integrated search bar, category filters, and an inline overlay article reader with scroll-progress indicator.
* 📬 **Simulated Contact Form**: Real-time form validations, submission loading spinners, and checkmark success animations.
* 💼 **Technologies Tab Showcase**: Slider and grid showcase for all my skills.
* 📄 **Direct Resume Downloads**: Easy download access for `vishal-resume.pdf`.

## 🚀 Getting Started

To run this project locally, clone the repository and use **Bun** (recommended for speed and matches lockfile configuration):

```bash
# 1. Clone the repository
git clone https://github.com/vishallrawal/portfolio.git
cd portfolio

# 2. Install dependencies
bun install

# 3. Run development server
bun run dev

# 4. Build for production compilation
bun run build

# 5. Preview production build
bun run preview
```
